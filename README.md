Pain of Paragon is a 2D side-scroller roguelike action game developed in Unity for PC. The
project focuses on responsive player movement, mouse-based combat, weapon and equipment
selection, armor and health management, enemy encounters, boss combat, run-level
progression, and upgrade selection.
